# Portfolio

Repository containing portfolio of example data science projects and links to certifications.

## Contents

- ### Data Science (including data analysis)
  - [NLP: Chat GPT Reviews](https://github.com/andycalcutt/Chat_GPT_Reviews): Pending Upload - Natural language processing analysis and predictive modelling of application reviews
  - [Time Series Forecasting: Store Sales](https://github.com/andycalcutt/Store_Sales): Pending Upload - Time series analysis and forecasting of store sales utiling SARIMA combination model
  - [Segmentation: Credit Card Customers](https://github.com/andycalcutt/Credit_Card_Customers): Pending Upload - Segmentation of customers based on behaviours and characteristics
  - [Classification: Titantic Survivors](https://github.com/andycalcutt/Titanic_Survivors): Pending Upload - Supervised Learning to predict survivors of the titanic based on features provided including age, gender, class

- ### Simulations
  - [ConnectX](https://github.com/andycalcutt/ConnectX): Bot utilising a minimax algorithm to play games of ConnectX which has the same principles as connect4 but the grid and in a row dimensions can be variable

- ### Object Orientated Programming
  - [Operations Monte Carlo Simulation](https://github.com/andycalcutt/Operations_Monte_Carlo_Simulation): Basic implementation of a monte carlo simulation to forecast completion timeframes of an operations process
